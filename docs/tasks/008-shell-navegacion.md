# 008 — Shell de navegación del demo

**Dueño:** A · **Costura S7** de [`plan.md §3`](../plan.md)

## Objetivo
Que el jurado recorra `/` → chat → `/asesor` → nutrición sin escribir una URL ni que nadie le explique dónde hacer clic.

## Alcance
- Dentro: layout común con acceso visible a la vista del asesor desde el chat y de vuelta a la landing.
- Dentro: en la landing, una línea que diga qué va a ver (los 3 personajes y qué representa cada uno).
- Fuera: diseño bonito. "Feo pero funciona" — el criterio es que se entienda sin narración, no que sea elegante.

## Done cuando
- [ ] Alguien ajeno al equipo recorre los 3 caminos completos sin preguntar nada.
- [ ] Ninguna pantalla del demo requiere escribir una URL a mano.

## Notas
A es dueño de `/`, D de `/asesor`; el layout que los une no era de nadie. Sólo depende de que ambas páginas existan, aunque sea contra fixtures.
